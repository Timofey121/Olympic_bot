## Сегодня олимпиады - это альтернативный способ поступить в вуз. Статус призера или победителя олимпиады  дает реальную возможность абитуриенту поступить в самые престижные вузы без вступительных испытаний или на преимущественных условиях (в зависимости от ВУЗа).
